# Trello Clone (Kanban Board)

A fullstack Kanban board inspired by Trello, built with a focus on clean structure, maintainability and real-world development practices.

## Tech Stack

Frontend
- React
- TypeScript

Backend
- Node.js
- Express

## Features

- Create and manage tasks
- Drag & drop between columns
- REST API
- Clean and modular structure

## Project Structure

client/   → Frontend  
server/   → Backend  

## Getting Started

### Clone

git clone https://github.com/kadlias/rello-clone-pro.git  
cd rello-clone-pro  

### Server

cd server  
npm install  
node index.js  

### Client

cd client  
npm install  
npm run dev  

## Notes

This project focuses on demonstrating fullstack fundamentals, clean structure and scalable architecture.
